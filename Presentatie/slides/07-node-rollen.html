Drie soorten kastjes

Repeater — het doorgeefluik. Stuurt berichten verder, dag en nacht.

Companion — je persoonlijke radio. Koppelt aan je telefoon via Bluetooth.

Room Server — het geheugen van het netwerk. Slaat berichten op zodat je ze later nog kunt lezen.

Bonus: een companion kan tijdelijk ook als repeater werken — ad-hoc modus. Handig onderweg!

Drie dingen onthouden: repeater, companion, room server. Meer is het niet. Base Station bestaat ook, maar die is voor gevorderden — niet noemen vanavond.

🔬 Verdieping: Repeater is headless, firmware-only. Companion maakt BLE-verbinding met MeshCore-app. Room Server draait MQTT-bridge en message store. Overgang: "Drie dingen onthouden: mesh, LoRa, en drie soorten kastjes. Dat is alles wat je nodig hebt. En nu bewijs — we gaan het live doen."
